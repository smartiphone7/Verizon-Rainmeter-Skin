# Verizon-Rainmeter-Skin
Rainmeter skin that displays Wi-Fi signal strength using signal bars and 2G/3G/4G/5G indicators based on strength. Includes Verizon Wireless branding and battery icon that changes. Similar layout to iOS status bar on iPhones.

# How to use
Download repo as .zip and place folders in your Rainmeter skins directory (usually ../Documents/Rainmeter/Skins)

# Disclaimer
AI was used to write some of the code as this was meant to be a quick personal project. You will need Rainmeter to use this skin which can be downloaded at https://www.rainmeter.net/
